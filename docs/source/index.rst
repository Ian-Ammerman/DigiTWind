Welcome to DTFOWT!
===================================

**DTFOWT** is a combination of edited versions of [ROSCO](https://github.com/NREL/ROSCO) and [OpenFAST](https://github.com/OpenFAST/OpenFAST) and new capabilities such as [WRP](https://github.com/doe-fowt-control/py-wrp) (not implemented yet), tailored for the development of digital-twin model for floating offshore wind applications.

.. note::

   This project is under active development.
